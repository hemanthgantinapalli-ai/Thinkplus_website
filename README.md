## üíª Key Features & Technical Details

The website is a multi-page platform (7 total HTML pages) demonstrating core modern web development practices.

### 1. Structure and Technology

* **Technology:** HTML5, CSS3 (Vanilla), and JavaScript (Vanilla).
* **Design:** Fully Responsive (Mobile-First approach ensured via media queries).
* **Dependencies:** Uses **Font Awesome** for icons and the **Poppins** Google Font.

### 2. EdTech Functionality Showcase

| Feature Page | Description & Technical Highlight |
| :--- | :--- |
| **Home (`index.html`)** | Clear marketing hierarchy, visually compelling Hero Section, and a strategically designed **CAT Preparation Roadmap**. |
| **Courses (`courses.html`)** | A three-tiered (Basic, Medium, Advanced) pricing model using distinct card layouts and a comparison table for clear value communication. |
| **Mock Test (`mocktest.html`)** | **Core JavaScript functionality:** Implements a short, functional quiz that processes user input, calculates the score, and provides instant, color-coded visual feedback upon submission. |
| **Rewards (`rewards.html`)** | **Gamification element:** Features a daily streak tracker and a **custom JavaScript animation** for the Gold Coin counter, enhancing the user experience. |
| **Contest (`contest.html`)** | Details on weekly challenges and rewards (scholarships/discounts) to drive competitive participation. |
| **Certificates (`certificates.html`)** | Outlines the value proposition of receiving verified course completion certificates (Basic, Medium, Advanced). |

## üé® UI/UX & Creative Decisions

The design choices were made to reflect the professionalism and engagement levels expected from a top EdTech brand:

* **Color Palette:** Use of a bold **Primary Blue (`#003366`)** for trust and corporate identity, contrasted with a **Vibrant Orange (`#FFA500`)** as an accent for calls-to-action and critical information.
* **Visual Depth:** Cards and feature boxes utilize subtle **box-shadows** and **hover states** (`transform: translateY(-8px);`) to give the interface a premium, polished feel.
* **Navigation:** Clean header navigation with full responsiveness, including a functional **mobile hamburger menu** built with JavaScript.
